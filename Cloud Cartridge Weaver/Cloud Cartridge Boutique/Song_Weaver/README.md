The Song Weaver (SONG_WEAVER.HTML) is an interactive musical environment that captures live melodies and "weaves" them into standalone, portable Cloud Cartridges. It is specifically designed to bridge the gap between playing by ear and reading formal sheet music.

Core Logic Features:

Live Capture: Record melodies instantly without complex DAW software.

Visual Staves: Automatically renders saved songs onto a 5-line G-Clef staff.

Animated Playback: Notes blink and highlight in real-time as they play to help students track rhythm and pitch.

Zero Footprint: A single file that lives on your hard drive and runs in any browser.

Offline Autonomy: No API key is required, and no internet connection is needed to run the file once it is saved to your device.
