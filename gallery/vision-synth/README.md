This program allows you to make images from text or edit images that you upload. It removes the "guesswork" by giving you direct control over the engine's output parameters:

Text-to-Image & Image-to-Image: Generate entirely new scenes from a description or upload a source photo to apply specific "Smart Edits."

Locked Dimensions: Choose between Square (1:1), Widescreen (16:9), or Vertical (9:16). The program forces the AI to stay within these exact bounds.

Cinematic Overrides: Includes specialized logic to force "Letterbox" (top/bottom) or "Pillarbox" (sides) black bars onto a square source, creating an instant cinematic movie-frame look.

Buffer Management: Features a "Wipe Buffer" command to clear the current session memory and start a fresh synthesis cycle.
